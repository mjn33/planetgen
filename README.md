# PlanetGen

A procedurally generated planetary surface using quad-trees.

## Controls

| Key    | Description            |
|--------|------------------------|
| W      | Move foward            |
| S      | Move backward          |
| A      | Move left              |
| D      | Move right             |
| Q      | Roll counter-clockwise |
| E      | Roll clockwise         |
| LShift | Move faster            |
| ,      | Slow down time         |
| .      | Speed up time          |
